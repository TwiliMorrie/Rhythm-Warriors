## üé® SKIN UNLOCK SYSTEM

### How to Get Skins:
1. **Story Rewards** - Unlock by completing chapters
2. **Achievement Rewards** - Complete specific challenges
3. **Coin Purchase** - Buy from Heroes screen
4. **Event Exclusive** - Limited time during events
5. **Ultimate Unlock** - Max out everything

### Skin Categories:
- **Common**: 0-1500 coins (early unlocks)
- **Rare**: 1600-2500 coins (mid-game)
- **Epic**: 2600-4000 coins (late-game)
- **Legendary**: 4000+ coins (endgame)
- **Mythic**: Event or ultimate unlocks

### Visual Effects by Rarity:
- **Common**: Color change, basic particles
- **Rare**: Animated effects, enhanced particles
- **Epic**: Unique animations, special SFX
- **Legendary**: Full visual overhaul, custom sounds
- **Mythic**: Screen effects, aura, ultimate visual identity

## üé≠ SKIN PROGRESSION PATH

### New Player:
1. Start with Rhythm Knight (default)
2. Earn Festival Champion skin (complete story)
3. Buy Shadow Dancer + Void Assassin skin (side quests)
4. Unlock event skins during festivals

### Mid-Game Player:
1. Purchase Iron Guardian + Runic Fortress
2. Unlock Pixel Crusader (1000 enemies)
3. Get Cyber Ninja (Chapter 5)
4. Collect rare skins

### Endgame Player:
1. Max all heroes and upgrades
2. Unlock all legendary skins
3. Complete all events for event skins
4. Work toward Champion of Harmonia (ultimate goal)
